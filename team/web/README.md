API 서버 execute ```bash cd team uvicorn api.main:app --reload --port 8000 ``` ### 2. 브라우저from 접속 ``` http://localhost:8000 ``` --- ## 📋 주요 feature ### Agents 탭 각 agent 개별적as executewill 수 exists. #### Project Planner - **input**: project select + requirements (텍스트 또 file) - **output**: ticket file create - **use법**: 1. project select 2. input approach select (텍스트 / file 업로드) 3. requirements input 또.md file 업로드 4. execute 버튼 클릭 #### PM Agent - **input**: project select + ticket select - **output**: specification 및 test 케스 - **use법**: 1. project select (ticket list auto 로드) 2. ticket select 3. execute 버튼 클릭 #### Coding Agent - **input**: project select + ticket select - **output**: code implementation - **use법**: 1. project select (ticket list auto 로드) 2. ticket select 3. execute 버튼 클릭 #### QA Agent - **input**: project select + ticket select - **output**: test code - **use법**: 1. project select (ticket list auto 로드) 2. ticket select 3. execute 버튼 클릭 #### Stack Initializer - **input**: project select - **output**: 초기 스택 구성 - **use법**: 1. project select 2. execute 버튼 클릭 --- ### Pipelines 탭 PM → Coding → QA all pipeline execute. **use법**: 1. project select (ticket list auto 로드) 2. ticket select 3. "Run Pipeline" 버튼 클릭 4. pipeline progress 상황 모니터링 **feature**: - 실시간 progress 상황 표시 - 각 step별 status (pending, running, success, failed) - progress률 (%) 표시 - pipeline 상세 info check (클릭) - 로그 조회 --- ### Skills 탭 재use 능one 스킬 execute. **지원 스킬**: - validate-spec: specification validation - commit: commit message create - review-pr: PR review - refactor-code: code refactoring - test-runner: test execute - deploy: deployment - benchmark: 성능 test - docs-generator: documentation create **use법**: 1. 스킬 카드of "Run" 버튼 클릭 2. ticket input (select 사항) 3. project input 4. check --- ### Statistics 탭 all 통계 및 최근 활동 check. **표시 info**: - Total Pipelines: all pipeline 수 - Success Rate: success률 (%) - Avg Duration: 평균 소요 시간 (초) - Active Now: 현재 execute 중인 pipeline 수 --- ## 🎨 UI/UX 특징 ### project/ticket select - **project**: Select box로 listfrom select - APIfrom auto 로드 (`/api/projects/list`) -
